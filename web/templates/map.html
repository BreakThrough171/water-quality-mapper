질소·인 유출 위험 지역 지도

공간 조인 기반으로 정확한 수질 현황을 시각화한 지도입니다. 각 행정구역의 수질 상태를 5단계 경보 시스템으로 분류하여 표시합니다.

📊 경보 단계별 색상 범례

1단계 (수질 우수)
1단계 (수질 양호)
2단계 (수질 보통)
3단계 (수질 나쁨)
4단계 (수질 매우 나쁨)
5단계 (수질 위험)

📋 지도 정보

🎯 평가 방법

총인(TP)과 총질소(TN) 값을 가중치 기반으로 계산하여 5단계 경보 시스템으로 분류합니다.

🗺️ 지도 범위

전국 17개 시도와 주요 시군구를 포함한 행정구역별 수질 현황을 표시합니다.

📊 데이터 소스

국립환경과학원 수질 DB의 실시간 측정 데이터를 기반으로 분석합니다.

🔄 업데이트

실시간으로 수질 데이터를 수집하고 분석하여 지도를 업데이트합니다.
